# CONTENT_PLAN

## 콘텐츠 전략 개요 (KR)
현재 랜딩 페이지의 섹션 흐름(히어로 -> 소개 -> 서비스 -> 프로세스 -> 문의)을 유지하면서, 메시지의 설득 순서와 CTA 선명도를 높이는 것을 목표로 한다.

## 섹션별 카피 매트릭스
| 섹션 | 사용자 질문 | 전달할 메시지 | 권장 CTA |
|---|---|---|---|
| Hero | "여기가 어떤 팀인가?" | HeptaLabs는 전략과 실행을 연결하는 파트너 | 상담 문의 / 서비스 보기 |
| About | "왜 신뢰할 수 있나?" | 비전, 실행력, 파트너십 기반 접근 | 우리 방식 보기 |
| Services | "무엇을 해주나?" | 소개 웹 구축, 전략 컨설팅, 운영 최적화 | 서비스 상담 |
| Process | "어떻게 진행되나?" | 진단 -> 설계 -> 실행 -> 고도화 | 프로세스 상담 |
| Contact | "어떻게 시작하나?" | 빠른 초기 상담과 맞춤 제안 가능 | 메일/전화 문의 |

## CTA 문안 가이드
- 1차 CTA(강한 행동): "상담 문의", "프로젝트 시작하기"
- 2차 CTA(탐색 행동): "서비스 보기", "프로세스 확인"
- CTA는 버튼 라벨 길이를 4~10자(한글 기준)로 유지해 스캔성을 확보

## 톤 앤 매너
- 신뢰 기반: 과장된 수식보다 근거 중심 문장
- 실행 중심: "할 수 있습니다"보다 "어떻게 진행합니다"를 먼저 제시
- 간결 우선: 한 문단 2~3문장 이내

## 카피 업데이트 원칙
1. 숫자/성과 수치는 근거 없는 경우 제거
2. 브랜드명 표기 통일: HeptaLabs
3. 문의 정보는 실제 운영 정보와 일치하도록 유지
4. 한 언어 수정 시 다국어 문서 동기화 여부 확인
